Print TAX Report PDF

Accounting Tax PDF Report Odoo App

This app allows your user to print a tax report in pdf format as shown in the below screenshots.

Main Features:

  • This app allows accounting users to print a tax report as shown below screenshots.
  • Users can configure taxes (Chart of taxes) and it will create the tax report hierarchy, and the hierarchy will be used in the print tax report.
  • We have added an option to display details (Accounting Journal Items) on the wizard which will allow users to analyze reports in detail.
  • This module allows the user to configure N number of levels for taxes and the user can view it on the tax report. Values on a tax report can be filtered also based on dates.
  • For more details please check the below screenshots and watch the video.

On tax report configuration we have provided options:

  • View
  • Taxes

This module allows the user to configure N number of levels for taxes and the user can view it on the tax report. Values on tax reports can be filtered also based on dates.
